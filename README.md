# Black_Friday_Sales_Analysis
Black Friday Sales Analysis with Python 550K+ transactions. SQL + pandas + seaborn.  From raw data to business insights.

Black Friday Sales Analysis
📌 Project Overview

This project focuses on analyzing Black Friday sales data to uncover meaningful insights about customer purchasing behavior, product trends, and sales patterns.

The dataset contains over 550,000+ rows, making it a large-scale data analysis project using multiple tools and technologies.

🎯 Objectives
Understand customer demographics and buying patterns
Identify top-selling products and categories
Analyze the impact of age, gender, and occupation on purchases
Perform data cleaning and preprocessing
Create visualizations to communicate insights effectively
🛠️ Tools & Technologies Used
Python (Jupyter Notebook)
Pandas – Data manipulation
NumPy – Numerical operations
Matplotlib & Seaborn – Data visualization
SQL – Data querying
Excel – Data cleaning & preprocessing
Power BI – Dashboard creation

Purchase Trend Analysis

📊 Key Insights
Certain age groups contribute more to total sales
Male customers tend to spend more compared to female customers
Some product categories dominate overall revenue
City category and occupation significantly affect purchase behavior
